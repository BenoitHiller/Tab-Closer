Tab Closer Settings

Help

Tab closer currently provides two exciting features for your enjoyment:

Predefined Matchers

On the right you can define several different patterns. When you click on the extension button the extensions will close all the tabs it can find that match the patterns you have put in.

Omnibar Tab Closing

New in this version is the abillity to close tabs using an omnibar command.

If you type 
c *spacebar*
into the omnibar you will be presented with the tab-closer prompt.

In this prompt you can type in a regular expression. When you press enter any tab that matches the expression, either in the title or in the url, will be closed.

Writing Matchers

By default you specify URL matchers in the following format: 
PROTOCOL://HOST/PATH
Where PROTOCOL can be any of:
  • *
  • http
  • https
  • file
  • ftp
The * is a wildcard and it will match any of the other options.

HOST is any valid address. You can use a *. at the begining only to match anything string to the left of the period.

PATH can be any string with wildcards anywhere within it.

You can check out Google's page on the topic for examples or a more thorough description.

Regex Matching

If you click the regex checkbox below a matcher it will allow you to specify any regular expression and close all tabs whose URLs match that.

If you don't know what a regular expression is, it is a language that is used for specifying search patterns.
